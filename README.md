# capJS
a javascript library that operates on capture files  

## Features
 - PCAP and PCAPNG Support
 - GZIP compressed files Support
 - Basic filters (select best, auth or not, etc...)
 - HCXDUMPTOOL Support
 - AP-LESS Passwords Feature
